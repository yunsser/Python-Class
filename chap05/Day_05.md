# Day_05

mode : 파일이 열리는 모드

| mode | 내용 |
| --- | --- |
| ‘r’ | 읽기 용으로 열림 (기본값) |
| 'w’ | 쓰기 위해 열기, 파일을 먼저 자른다. |
| 'x’ | 베타적 생성을 위해 열리고, 이미 존재하는 경우 실패 |
| 'a’ | 쓰기를 위해 열려 있고, 파일의 끝에 추가하는 경우 추가합니다. |
| 'b’ | 2진 모드(바이너리 모드) |
| 't’ | 텍스트 모드 (기본값) |
| '+’ | 업데이트 (읽기 및 쓰기)를 위한 디스크 파일 열기 |
| 'U’ | 유니버설 개행 모드 (사용되지 않음) |
